## Установка и запуск

Зависимости:

```bash
# Можно использовать любой пакетный менеджер, но я предпочитаю pnpm
pnpm install
```

Dev-сервер на `http://localhost:3000`:

```bash
pnpm run dev
```

Production-сборка:

```bash
pnpm run build
```
